WINDOW_WIDTH = 1080 - ширена окна программы 
FONEREGISTER = false - разрешает / заприщяет перевод работы в трей или в фоне, false - как поумолчанию заприщяет, true - как разришение
DROP_TICK = 150 - На каком тике сбросить работу программы
TIME_TICK = 3000 - время обновления Тика в милисекундах 

------ Второстепенные 
VERSION = 0.1+ - устонавлевает границу версии программы, понадаметься если нужно отключить контроль версий.
ERROR_PROCCES_CLOUSE = true - закрывать программу в случай ошибки true - как да, все остольные пораметры будут восприменяться как false
TOPMOST  = false программа всегда открыта поверх всех окон false как по умолчанию
